# 🚀 Knowledge Graph vs Traditional RAG – AI Chatbot System

Excited to share a project I worked on 🚀  

This project demonstrates a comparison between Traditional RAG (Retrieval-Augmented Generation) and a Knowledge Graph-based AI system.

## 💡 What this application does

This application allows users to ask questions about a recruitment system using natural language.

Example:
“What services does SIS International provide?”

## 🎯 How it works

### Traditional RAG
- Document chunking
- Embeddings + FAISS
- Retrieval-based answers

### Knowledge Graph
- Entity extraction
- Neo4j relationships
- Contextual reasoning

## ⚙️ Tech Stack
Python, Neo4j, LangChain, FAISS, OpenAI, PyVis, Matplotlib

## 📊 Features
- Chatbot
- RAG vs KG comparison
- Graph visualization
- Performance charts

## 🧠 Learning
RAG retrieves. Knowledge Graph understands.

## 🙌 Closing Note
Moving from retrieval to reasoning is the future of AI 🚀
